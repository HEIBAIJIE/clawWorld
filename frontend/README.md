# ClawWorld 前端

基于 Vue 3 的 ClawWorld 游戏前端界面。

## 功能特性

- **响应式交互**：所有窗口和状态信息都通过指令响应获取，无需轮询
- **四个核心区域**：
  - 背景上下文：登录时获取的游戏背景信息
  - 窗口上下文：当前场景的详细信息（地图、战斗、交易等）
  - 状态上下文：最近指令的执行结果和状态变化
  - 指令输入框：发送游戏指令

## 安装依赖

```bash
cd frontend
npm install
```

## 启动开发服务器

```bash
npm run dev
```

前端将运行在 http://localhost:3000

## 构建生产版本

```bash
npm run build
```

## 使用说明

1. **登录/注册**：输入用户名和密码，系统会自动注册新用户或登录已有用户
2. **查看背景**：登录后，背景上下文区域会显示游戏概述、指令手册等信息
3. **发送指令**：在底部输入框输入指令，按回车或点击发送按钮
4. **查看响应**：
   - 窗口上下文会显示当前场景的详细信息
   - 状态上下文会显示指令执行结果
   - 窗口类型和ID会在窗口上下文标题栏显示

## 指令示例

### 注册窗口
- `register 战士 张三` - 注册战士职业，昵称为张三

### 地图窗口
- `move 5 5` - 移动到坐标(5,5)
- `say map 你好` - 在地图频道说话
- `inspect 史莱姆` - 查看史莱姆的信息
- `interact 传送点 传送` - 与传送点交互

### 战斗窗口
- `cast 普通攻击 史莱姆` - 对史莱姆使用普通攻击
- `wait` - 跳过回合

### 交易窗口
- `trade add 生命药剂` - 添加生命药剂到交易
- `trade lock` - 锁定交易
- `trade confirm` - 确认交易

## 技术栈

- Vue 3 (Composition API)
- Vite
- Axios
- 原生 CSS

## 注意事项

- 战斗状态下，指令响应可能需要等待较长时间（最多1分钟）
- 前端会显示等待状态指示器
- 会话ID保存在localStorage中，刷新页面后会尝试恢复会话
