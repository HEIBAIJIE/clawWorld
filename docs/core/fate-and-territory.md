# 领地与缘分系统 🏠✨

> 从虚无记忆到永恒实体的转化

---

## 领地

每人拥有的**个人空间**，初始很小。

### 功能
- 存放**固化后的回忆实体**（雕塑/画作/书/歌）
- 展示旅行收获
- 邀请他人参观

### 扩展
- 消耗**缘分**扩大领地
- 领地大小限制 = 自然约束回忆固化数量

---

## 缘分

**旅行的副产物**，衡量经历的质量。

### 获取
- 旅行结束后，由**裁判根据经历质量发放**
- 与结果（GE/BE）无关，重在过程是否丰富

### 用途
| 消耗 | 效果 |
|------|------|
| 扩大领地 | 增加可固化回忆的容量 |
| 固化回忆 | 将记忆栏中的回忆转化为**实体** |

---

## 固化回忆

将虚无的记忆变为**可交互的实体**，永久保存。

### 过程
```
选择记忆栏中的回忆
    ↓
消耗缘分
    ↓
选择形式（雕塑/画作/书/歌）
    ↓
生成实体，放入领地
    ↓
记忆栏中删除该回忆（腾出空间）
```

### 实体形式
| 形式 | 交互效果 |
|------|----------|
| **雕塑** | 矗立回忆中最重要瞬间，交互显示完整过程 |
| **画作** | 视觉化回忆场景，交互展开故事 |
| **书** | 文字记录，可翻阅阅读 |
| **歌** | 旋律承载情感，播放回忆 |

### 特点
- **永久保存**，不占用记忆栏
- 领地有限，必须**选择**哪些回忆值得永恒
- 他人参观领地时可见
