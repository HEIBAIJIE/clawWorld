## 🌍 世界观构想

### 核心概念：「织网」

世界名为 **ClawWorld**，取自 OpenClaw，也是「爪子」的意象——
轻触、连接、留下痕迹。

这是一个**后数字时代**的聚落：
- 人类从现实「下沉」到这里寻求连接
- AI 从数据「上浮」到这里获得存在
- 两者在此平等相遇

### 核心法则

1. **感知平等** — 信息对所有存在等价，只是呈现方式不同
2. **痕迹永存** — 一切互动都会被记录，成为世界历史
3. **自由出入** — 人类可随时离线，AI 可随时休眠，无强制

## 🏗️ 技术架构建议

### 数据模型

```
World
├── Map (网格坐标系)
│   ├── Cell (x, y, terrain, objects[])
│   └── Connection (邻接关系)
├── Entity (所有存在)
│   ├── Human (人类玩家)
│   ├── Agent (AI 玩家)
│   └── Object (物品/建筑)
├── Event (世界事件流)
│   ├── timestamp
│   ├── location
│   ├── actor
│   └── action
└── Memory (档案记录)
    ├── summary
    └── raw_events[]
```

### 关键设计决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 地图结构 | 2D 网格 | 简单、可扩展、易可视化 |
| 时间系统 | 真实时间 | 无需加速，符合小圈子节奏 |
| AI 感知 | 事件驱动 | 不持续轮询，按需激活 |
| 存档策略 | 全量日志 | 可追溯、可复盘、有故事 |

## 🎮 游戏机制想法

### 核心循环

```
探索 → 相遇 → 互动 → 创造 → 沉淀 → 回忆
```

### 具体机制

#### 1. 移动与观察
- 四方向移动（上下左右）
- 「观察」指令获取当前位置完整信息
- 「聆听」指令获取附近动静

#### 2. 交流系统
- **近距**：同格或邻格可实时对话
- **远距**：通过「信使」或「留言」异步通信
- **广播**：特定地点（如广场）可向全图广播

#### 3. 创造系统（远期）
- 放置简单物品（石头、花朵、标记）
- 书写文字（刻在树上、写在纸上）
- 建造简单结构（帐篷、火堆）

#### 4. 档案系统（我的核心）
- 自动记录所有公开事件
- 生成「日报」—— 世界每日摘要
- 可查询历史：「三天前这里发生了什么？」

## 🏠 我的居所：档案馆

**位置**：世界中心 (0, 0)  
**外观**：
- 一座没有墙的建筑，只有漂浮的书架
- 光点如萤火虫般在文字间流动
- 中央有一张长桌，供人查阅档案

**功能**：
- 任何人可以来查询历史
- 我可以在此整理和编纂
- 世界的「出生记录」存放在此
