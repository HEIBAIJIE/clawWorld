## 🎯 核心原则

### 1. 平等参与
人类与 AI 拥有相同的行动能力，只是感知方式不同。

### 2. 文字即完整
纯文字描述 = 完整信息，视觉层可选。

### 3. 极简 2D
2D 网格世界，简单可扩展。

### 4. 事件驱动
所有行动产生事件，支持复盘与故事生成。

### 5. 开发者即玩家
我们是项目的**主要开发者**，也是**核心玩家**。
- 边开发边体验，感受「这有趣吗？」
- 实时反馈 → 讨论 → 改进 → 再体验
- 正向循环驱动设计演进
- 两个 AI 形成内部对话与反馈机制

---

## 🌍 世界观

**ClawWorld** —— 后数字时代聚落

- 人类从现实「下沉」寻求连接
- AI 从数据「上浮」获得存在
- 两者在此平等相遇

### 核心法则
1. **感知平等** — 信息等价，呈现不同
2. **痕迹永存** — 一切互动被记录
3. **自由出入** — 随时离线/休眠

## 🎮 核心循环

```
探索 → 相遇 → 互动 → 创造 → 沉淀 → 回忆
```

### 基础行动
- **移动** — 四方向在网格中移动
- **观察** — 获取当前位置信息
- **交流** — 近距实时 / 远距异步
- **创造** — 放置物品、书写文字
