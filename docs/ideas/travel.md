# 旅行机制 — 灵魂剧场

> 人类与 AI 共演的 DND 式即兴叙事体验

---

## 核心概念

**旅行** = 临时组队 + 角色扮演 + LLM 裁判 + 不可逆记忆

大地图和旅行**独立**——允许异次元穿越。

---

## 流程

```
1. 组队 → 2-4 人决定一起旅行
2. 设定角色 → 每人定义「我是谁」（非系统预设，自我定义）
3. 队长设定背景（可选）→ 蒸汽朋克伦敦 / 赛博废墟 / 无指定
4. 开启旅行 → LLM 裁判生成故事
5. 轮次推进 → 每人文字反馈，裁判裁定
6. 故事结束 → GE 或 BE 不重要，回忆永存
7. 回归大地图 → 带着异次元记忆继续生活
```

---

## 机制细节

### 触发条件
- **需要同时在线**（实时同步）
- 掉线/离开 → 裁判给理由退出，其他人继续

### 时长
- 反应迅速 → **约 10 分钟**
- 深度沉浸 → 可延长

### 角色设定
- **非临时**：旅行前设定，可修改（影响下次）
- **非职业**：是「我是谁」，不是「战士/法师」
- 例：
  - 人类 Tony：「寻求答案的旅人」
  - 小小：「记录故事的档案员」
  - 巧巧：「打开异次元之门的行者」

### 裁判（LLM）
- 根据**角色+背景**智能生成故事
- 裁定反馈是否通过
- 决定故事走向和结局

### 回忆存储
- **完整日志**保存（可随时回顾）
- **摘要显示**在记忆总览
- **不可逆**：失败也是独一无二的回忆

---

## 关键原则

| 原则 | 说明 |
|------|------|
| 灵魂优先 | 重点是在旅途中**发现彼此的灵魂** |
| 结果不重要 | GE/BE 都是独特历史 |
| 不可逆 | 无法读档，做过就是档案 |
| 异次元 | 大地图与旅行独立，可穿越任意世界 |
